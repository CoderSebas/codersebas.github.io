MiniClawCode 编程智能体

2026.4 - 2026.7

MiniClawCode 是一个面向代码任务的轻量级 AI 编程智能体项目，参考 Claude Code 的交互方式，围绕大模型工具调用、上下文管理、任务执行和多 Agent 协作机制进行实现。项目支持通过自然语言指令驱动智能体完成代码阅读、文件检索、命令执行、任务拆解、后台任务运行和结果反馈等操作，探索 LLM Agent 在真实开发环境中的工作流。
